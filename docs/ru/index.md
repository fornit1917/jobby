features:
  - title: Консистентность
    details: Фоновые задачи хранятся в базе данных. Поддерживается транзакционное создание нескольких задач и создание задач в одной транзакции с вашими бизнес-операциями.
  - title: Функциональность
    details: Мульти-очередь задач, задачи по расписанию, конфигурируемый порядок исполнения, гибкие политики повторов для упавших задач, OpenTelemetry-совместимые метрики и трейсинг и многие другие возможности доступны из коробки.
  - title: Расширяемость
    details: Простая интеграция с различными ORM, возможность использовать собственные планировщики для задач по расписанию, настраиваемый middlewares-пайплайн исполнения задач.
  - title: Масштабируемость
    details: Поддерживается параллельное исполнение задач как в одном инстансе, так и на нескольких инстансах в распределенной системе.
  - title: Надёжность и отказоусточивость
    details: Автоматическое обнаружение выбывших инстансов и распределение их задач среди остальных. При недоступности базы данных корректно ожидается восстановление, операции повторяются без потери данных.
  - title: Высокая производительность
    details: Работает в несколько раз быстрее аналогичных библиотек (таких как Hangfire или Quartz) и потребляет в несколько раз меньше ресурсов как на стороне .NET-приложения, так и на стороне базы данных.
